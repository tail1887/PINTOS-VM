## Merge 폴더 규칙

- 폴더명: `Merge N - …` (원문 장 제목과 동일)
- 그 안 **`00-서론.md`**: 해당 장 **§N.1**(목표·**N.1.1** E2E·**N.1.2** 요약)·**§N.2**(내부 머지 순서)·**§N.7**(완료 기준)
- **`A - …` ~ `D - …`**: 분업 A~D 본문(시퀀스·단계 포함)

이전 **단일 파일 전체**는 git 히스토리에서 복원할 수 있다.
