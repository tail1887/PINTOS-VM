# 03 — `args-multiple` 검증

## 1. 이 테스트가 검증하는 것
여러 인자 전달 시 순서 보존과 `argv` 인덱싱 정확성을 검증합니다.

## 2. 구현 필요 함수와 규칙
### 인자 파싱/저장 루틴
- 위치: `pintos/userprog/process.c`
- 규칙 1: 토큰 순서를 유지한다.
- 규칙 2: 중간 인자가 누락되지 않게 개수와 배열 인덱스를 함께 갱신한다.

### 스택 포인터 조정 루틴
- 위치: `pintos/userprog/process.c`
- 규칙 1: 문자열 구간과 포인터 구간이 겹치지 않게 경계 체크를 한다.
- 규칙 2: 최종 포인터가 사용자 주소 범위를 벗어나지 않는다.

## 3. 실패 시 바로 확인할 것
- `argv[i]`가 이전/다음 인자와 뒤바뀌는지
- `argv[argc]` 널 센티널이 빠졌는지

## 4. 재검증
- `args-multiple` 단일 실행 통과 확인
