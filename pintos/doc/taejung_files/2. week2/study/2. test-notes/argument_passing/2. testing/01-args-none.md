# 01 — `args-none` 검증

## 1. 이 테스트가 검증하는 것
인자가 없는(또는 프로그램 이름만 있는) 최소 입력에서
프로세스 시작 경로가 안정적으로 동작하는지 검증합니다.

## 2. 구현 필요 함수와 규칙
### `process_exec()`
- 위치: `pintos/userprog/process.c`
- 규칙 1: 토큰 수가 최소여도 정상적으로 로드 경로를 진행한다.
- 규칙 2: `argc`와 `argv`를 일관되게 설정한다.

### 스택 인자 배치 루틴
- 위치: `pintos/userprog/process.c`
- 규칙 1: `argv[argc] == NULL`을 보장한다.
- 규칙 2: 가짜 return address를 포함한 기본 프레임을 구성한다.

## 3. 실패 시 바로 확인할 것
- `argc` 값이 0/1 경계에서 의도와 다르지 않은지
- `argv`가 NULL이거나 잘못된 사용자 주소를 가리키지 않는지

## 4. 재검증
- `args-none` 단일 실행 통과 확인
