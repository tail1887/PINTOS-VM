# 04 — `args-many` 검증

## 1. 이 테스트가 검증하는 것
인자 개수가 많은 경우에도 스택 레이아웃과 포인터 배열이 안정적으로 유지되는지 검증합니다.

## 2. 구현 필요 함수와 규칙
### 인자 제한/경계 처리
- 위치: `pintos/userprog/process.c`
- 규칙 1: 한 페이지 한도를 넘기지 않도록 총 사용 바이트를 체크한다.
- 규칙 2: 한도 초과 시 부분 상태를 정리하고 실패 반환한다.

### 스택 구성 루틴
- 위치: `pintos/userprog/process.c`
- 규칙 1: 문자열 복사 + 포인터 push + 정렬이 순서대로 적용된다.
- 규칙 2: 마지막에 `argc`, `argv`, fake return address가 모두 존재한다.

## 3. 실패 시 바로 확인할 것
- 많은 인자에서만 페이지 폴트가 나는지
- 경계 근처에서 `argv` 주소가 깨지는지
