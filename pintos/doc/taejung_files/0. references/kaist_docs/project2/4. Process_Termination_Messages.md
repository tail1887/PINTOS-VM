# Process Termination Messages

원문: [Process Termination Messages](https://casys-kaist.github.io/pintos-kaist/project2/process_termination.html)

## Process Termination Message

**프로세스 종료 메시지를 출력합니다.**

사용자 프로세스가 `exit`를 호출해 종료하든, 다른 이유로 종료되든 **항상** 해당 프로세스의 이름과 종료 코드를 출력하세요. 형식은 다음을 `printf`로 찍은 것과 같아야 합니다.

```c
printf ("%s: exit(%d)\n", ...);
```

출력하는 이름은 `fork()`에 넘긴 **전체 이름**이어야 합니다. 사용자 프로세스가 아닌 커널 스레드가 종료될 때나, halt 시스템 콜이 호출될 때는 이 메시지를 출력하지 마세요. 적재에 실패한 프로세스에 대해서는 메시지가 **선택**입니다.

이 외에 Pintos 기본 제공이 이미 출력하지 않는 **추가 메시지**는 출력하지 마세요. 디버깅에는 유용할 수 있지만 채점 스크립트를 혼란스럽게 해 점수가 깎일 수 있습니다.
