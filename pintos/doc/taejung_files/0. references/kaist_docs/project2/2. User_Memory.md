# User Memory

원문: [User Memory](https://casys-kaist.github.io/pintos-kaist/project2/user_memory.html)

## User Memory Access

**사용자 메모리 접근을 구현합니다.**

시스템 콜을 구현하려면 사용자 가상 주소 공간에서 데이터를 읽고 쓰는 방법이 필요합니다. 인자 값 자체를 읽을 때는 이 능력이 꼭 필요하지 않을 수 있습니다. 하지만 **시스템 콜 인자로 넘어온 포인터**가 가리키는 데이터를 읽을 때는 이 기능을 **거쳐서** 처리해야 합니다. 다소 까다로울 수 있습니다. 사용자가 잘못된 포인터, 커널 메모리를 가리키는 포인터, 또는 그 둘에 걸쳐 있는 블록을 넘기면 어떻게 할까요? 이런 경우에는 사용자 프로세스를 종료하면 됩니다.
