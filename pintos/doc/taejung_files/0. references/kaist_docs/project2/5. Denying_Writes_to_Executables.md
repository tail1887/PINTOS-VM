# Denying Writes to Executables

원문: [Denying Writes to Executables](https://casys-kaist.github.io/pintos-kaist/project2/deny_write.html)

## Deny Write on Executables

**실행 중인 실행 파일에 대한 쓰기를 거부합니다.**

실행 파일로 사용 중인 파일에 대한 쓰기를 막는 코드를 추가하세요. 디스크 위에서 코드가 바뀌는 도중에 실행하면 결과를 예측하기 어렵기 때문에 많은 OS가 이렇게 합니다. 프로젝트 3에서 가상 메모리를 구현하면 특히 중요하지만, 지금 막아 두어도 나쁘지 않습니다.

`file_deny_write()`로 열린 파일에 대한 쓰기를 막을 수 있습니다. `file_allow_write()`를 호출하면 다시 쓰기가 허용됩니다(다른 opener가 쓰기를 막고 있으면 예외). 파일을 닫아도 쓰기가 다시 허용됩니다. 따라서 프로세스가 실행 중인 동안에는 **실행 파일을 열어 둔 채** 쓰기를 거부해야 합니다.
